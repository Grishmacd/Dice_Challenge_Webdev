# Dice_Challenge_Webdev

This project is a simple **two-player dice game** built using **HTML, CSS, and JavaScript**. Every time you refresh the page, two dice are rolled randomly and the winner is displayed on the screen.

---

## What this project is

A lightweight browser-based dice game where:
- Player 1 gets a random dice value (1–6)
- Player 2 gets a random dice value (1–6)
- The page displays:
  - **Player 1 Wins**
  - **Player 2 Wins**
  - **Draw**

The game result updates on **every refresh**. :contentReference[oaicite:0]{index=0}

- Random dice roll generation using `Math.random()` (1 to 6) :contentReference[oaicite:4]{index=4}
